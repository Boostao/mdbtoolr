Create an mdbtoolr Driver

Description

Create a DBI driver backed by command line utilities from mdbtools.

Usage

Mdb()

mdb()

Value

An object of class MdbDriver.

Examples

if (nzchar(Sys.which("mdb-tables"))) {
  drv <- Mdb()
}
